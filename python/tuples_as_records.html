tuples as records

  1. 19 April 2017
  2. python
  3. general

tuples can be used as simple records.

volunteers = [('Bob', 36, 'M'), 
              ('Alice', 27, 'F'), 
              ('Eve', 41, 'F')]

for hero in sorted(volunteers):
    print('%s/%d/%s' % hero)

Alice/27/F
Bob/36/M
Eve/41/F
